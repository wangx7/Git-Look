# Git Look

一个优雅的 VS Code 插件，提供可视化的 Git 历史图表和精准的代码源头修改历史追溯。

## 功能

1. **Git 历史图表面板 (底部 Panel)**：
   - 交互式分支图渲染（使用 SVG 和贝塞尔曲线）。
   - 按分支、作者、日期范围、文本或 commit 哈希过滤。
   - 在图表列表中点击提交查看变动文件列表，并可通过原生 diff 视图对比修改。

2. **代码源头与修改历史追溯 Tab 页**：
   - 选中行/区域后右键选择“追溯代码源头”。
   - 在编辑器中打开全新 Tab 页，以时间轴卡片形式展示修改历史。
   - 最底部的卡片标记为 **“最初创建”** 且默认展开，帮助您一眼锁定最初的开发者和提交记录。
   - 卡片内提供高保真 Diff 展示（左红右绿，空位填充灰色斜线条纹背景）。
   - 卡片带有“在图表中定位”按钮，可与底部 Git 历史面板进行定位联动。

## 安装

可以通过扩展市场的 VSIX 进行安装。
